# workstation-linux-config
Ansible playbook for workstation setup

# Requirements:
```
  - RedHat base distro (Alma, Rocky)
  - Gnome desktop
```
---
  
# How to use it:
```bash


cd workstation-linux-config
ansible-playbook workstation-linux-config -K
```
